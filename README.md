# YogaLíf

A modern, responsive website for YogaLíf - a yoga and pilates studio located in Kópavogur, Iceland.

## Features

- Responsive design that works on all devices
- Modern UI with smooth animations
- Multiple pages including:
  - Home page with class information
  - Teachers (Kennarar) page
  - Courses (Námskeið) page  
  - About (Um YogaLíf) page
  - Shop (Versla) page
  - KAP (Kundalini Activation Process) page
- Contact form
- Beautiful gradient color scheme

## Technologies Used

- HTML5
- CSS3 with CSS Variables
- Vanilla JavaScript
- Google Fonts (Playfair Display & Inter)

## Local Development

To run this website locally:

1. Clone the repository:
```bash
git clone https://github.com/Dannalingur/yogalif.git
```

2. Open `index.html` in your web browser

## Deployment

This website is configured for deployment on Render as a static site.

### Deploy to Render

1. Fork or clone this repository to your GitHub account
2. Sign up for a [Render account](https://render.com)
3. Create a new Static Site on Render
4. Connect your GitHub repository
5. Render will automatically deploy your site

The site includes a `render.yaml` configuration file for easy deployment.
